# Week03 — Git exercises

Инструкция:
1. Создать ветку `exercise/git-practice`.
2. Сделать 3 небольших коммита (изменить README, добавить файл, затем изменить его).
3. Сформировать локальный PR-описание (или открыть PR, если репозиторий на GitHub).

Цель: попрактиковаться в ветках, коммитах и описательных сообщениях.
